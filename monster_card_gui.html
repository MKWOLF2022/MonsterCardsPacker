MONSTER CARDS .card GUI Unpack / Repack Tool

브라우저 안에서만 작동하는 단일 HTML 도구입니다. 카드 파일은 외부 서버로 업로드되지 않습니다.

룰과 매너를 지켜 듀얼하자! 이 프로그램을 상업적인 용도로 이용하지 말아주세요.! 꼭 배려를 해서 이용해주세요! @MKWOLF2022

중요: 원본 .card 파일은 반드시 백업하세요. 이 도구는 사용자가 제공한 ES3 설정 기준으로 만든 비공식 편집 도구입니다. 게임 업데이트로 저장 포맷이 바뀌면 수정이 필요할 수 있습니다. 지나친 사용은 게임의 재미를 저해할 수 있습니다. 꼭 배려를 하면서 사용해주세요. 원작자 요청시 비공개됩니다.

1. 파일 열기

ES3Defaults.asset 가 없어도 기본 입력된 Password로 작업을 진행합니다.

Password가 변경된 경우 ES3Defaults.asset 파일을 넣어주세요.

혹은 .card 파일을 선택한 뒤 “카드 열기/분석”을 누르세요.

ES3Defaults.asset을 넣으면 encryptionPassword를 자동으로 읽습니다. 수동 입력값이 비어있지 않으면 이 값을 사용합니다.

상태

대기 중입니다.

2. 카드 정보

상태아직 카드가 열리지 않았습니다.
지원하는 구조 
ES3 JSON format=0
AES-CBC encryptionType=1
Gzip compressionType=1
PBKDF2-HMAC-SHA1(password, salt=IV, iterations=100, key=16 bytes)
.card 앞 16바이트 = IV
image1Bytes/image2Bytes/image3Bytes = base64 이미지 바이트

3. 이미지 확인 / 교체

카드가 열리면 image1Bytes, image2Bytes, image3Bytes 미리보기가 표시됩니다. 각 레이어에 새 PNG/JPG/WEBP 파일을 넣고 repack할 수 있습니다.

4. Unpack 다운로드

현재 열린 카드의 card.json, 이미지 파일, manifest.json을 ZIP으로 저장합니다.

5. JSON 고급 편집 / Repack

일반적으로 이미지만 바꿔서 사용하시면 됩니다. 카드 이름/수치까지 직접 바꾸려면 JSON을 수정한 뒤 repack하세요. 단, JSON 구조를 망가뜨리면 게임에서 열리지 않을 수 있습니다.
