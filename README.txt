MonsterCardsPacker / 20260611 ver
==============================================================

Files
-----
index.html       Main HTML structure
styles.css       UI styles
app.js           Tool logic: ES3 decode/encode, image preview, ZIP, repack
translations.js  Korean/English translation table

How to run
(사용방법법)
----------
1. Keep all four files in the same folder.
(위의 네 파일을 모두 같은 폴더에 놓습니다.)
2. Open index.html with the latest Chrome or Edge.
(크롬이나 엣지를 통해 index.html를 실행합니다.)
3. Select a .card file.
(.card의 확장자를 가진 카드 파일을 선택합니다.)
4. Click Open / Analyze Card.
(카드 열기/분석 을 눌러주세요. )
5. Replace image1Bytes/image2Bytes/image3Bytes if needed.
(이미지 파일 1 / 2 / 3 을 변경해주세요. PNG를 권장합니다.)
6. Click Download new .card.
(새 .card 다운로드를 눌러주세요.)

If the password used to unlock a ".card" file has been changed,
you may need to select the ES3Defaults.asset file to extract the password.
(단, ".card 확장자의 파일"을 해제하는 비밀번호가 변경됐을 경우
ES3Defaults.asset 파일을 선택하여 암호를 추출해야할 수 있습니다.)

Language support
----------------
Use the language selector in the upper-right corner.
Currently supported languages:
- Korean (한국어)
- English

To add another language later, edit translations.js and add a new option in index.html.

Notes
-----
This tool processes files locally inside the browser. Files are not uploaded to any server.
Always back up the original .card file before using a repacked result in the game.
